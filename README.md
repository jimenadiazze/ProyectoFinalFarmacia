Proyecto Final – Sistema de Farmacia (ASP.NET WebForms)
Este proyecto es un sistema básico de gestión para una farmacia, desarrollado en ASP.NET
WebForms y C#, utilizando una arquitectura sencilla basada en capas.
Características principales
• Gestión de Productos
- Agregar nuevos productos
- Editar información existente
- Eliminar productos
- Buscar productos por nombre o ID
• Interfaz Web (ASP.NET)
- Formularios .aspx
- Code-behind en C#
- Validaciones básicas
- Diseño simple y funcional

 Arquitectura en capas
FarmaciaFinal: Proyecto Web con páginas ASPX, estilos, scripts y configuración.
ModeloFar: Biblioteca con modelos como Producto.cs y Conexion.cs.
Tecnologías utilizadas
ASP.NET WebForms, C#, SQL Server, HTML/CSS/JS, Visual Studio.
Estructura del proyecto

Cómo ejecutar el proyecto
1. Abrir FarmaciaFinal.sln en Visual Studio.
2. Restaurar paquetes NuGet si es necesario.
3. Verificar la cadena de conexión en Web.config.
4. Ejecutar con IIS Express.
